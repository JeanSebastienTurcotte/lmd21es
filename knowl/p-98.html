Paragraphe

Pour l'utilisation de cocalc, voici la marche à suivre:

  1. Créez un compte sur cocalc.com.

  2. Créez un nouveau projet sur Cocalc, possiblement intitulé PreText.

  3. Pour créer un projet, entrez pretext new -i "MonPremierLivre" dans le terminal. L'option facultative -i vous demandera le titre de vos chapitres de manière interactive. Il sera possible de modifier, ajouter ou enlever des chapitres plus tard.

  4. De retour dans les fichiers du projet Cocalc, un dossier « monpremierlivre » devrait être créé. À l'intérieur, un dossier « source » contient le fichier main.ptx dans lequel on va rédiger le contenu.

  5. Éditez le fichier main.ptx afin d'ajouter le contenu souhaité.

  6. Dans le terminal, tapez cd monpremierlivre afin de déplacer l'espace de travail dans le bon dossier. Ensuite, tapez pretext build html pour compiler le manuel.

    Dans le dossier monpremierlivre, il devrait maintenant y avoir un dossier output qui contient un dossier html. Dans ce dernier, on retrouvera les fichiers composant le manuel. On peut ouvrir le fichier index.html afin de visualiser le résultat.
